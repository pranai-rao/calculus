Parametric Functions

We know that for a relation to be a function, it must pass the vertical line test—showing that each \(x\)-value is paired with only one \(y\)-value.

But this limitation excludes any curves with self-intersections or even some basic curves such as circles.

Parametrization is an alternative method for describing curves that is not subject to the severe restriction of the vertical line test.

Parametric functions are used to describe motion in a plane (or even in space) and is an underlying concept for multivariable calculus.


Normally, when we think of curves, we think of functions where the value of the \(y\)-coordinate is determined by the value of the \(x\)-value.

With parametric functions, both the \(x\)- and \(y\)-coordinates are determined by the value of the parameter—most often \(t\).

Eliminating the Parameter

Eliminating the parameter is a method of switching back to rectangular coordinates.

Given a parametric equation $$\begin{cases}x=f(t) \\ y=g(t)\end{cases}$$ we can eliminate the parameter in three ways.

  1. Solve each equation for the parameter \(t\), then set the equations equal to one another.
  2. Solve one equation for the parameter \(t\), then plug that value into the second equation.
  3. Solve each equation for part of a trigonometric identity, then plug both values into the identity.
